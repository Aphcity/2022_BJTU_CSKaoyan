北京交通大学在2022年考研将计算机专业相关的课程从之前 923 操作系统 / 925 数据结构 两门课程二选一的专业课测试模式调整为 计算机专业基础 一门大类课程。分别包括 925-数据结构（90分）和 923-操作系统（60分）的两门专业课测试。

本仓库主要记录了 北交考研“计算机专业基础”课程的一些真题资料、笔记、思维导图以及算法源码。
